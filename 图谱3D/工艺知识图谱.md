## 一、实体类型（Entity Types）

共 **4 大类实体**，对应图谱中不同颜色节点：

### 1. 核心工艺（core）
- **颜色**：青色（#6eb5ff）
- **节点大小**：最大（半径 1.8）
- **实体**：
  - 水泥生产总工艺 - 水泥生产核心流程：两磨一烧，包括生料制备、熟料煅烧、水泥粉磨三大阶段

### 2. 工艺流程（process）
- **颜色**：亮青色（#00d4ff）
- **节点大小**：中等（半径 0.85）
- **实体列表**：

| 实体名称 | 描述 |
|---------|------|
| 原料破碎 | 对石灰石、黏土、砂岩、铁矿石等原料进行破碎至合适粒度 |
| 原料预均化 | 通过平铺取料降低成分波动，保证原料成分稳定 |
| 生料粉磨 | 将配比后原料研磨至符合要求的生料粉末 |
| 生料均化 | 使生料成分均匀稳定，为煅烧提供基础 |
| 预热分解 | 五级旋风预热+分解炉，完成碳酸盐分解 |
| 回转窑煅烧 | 1450℃左右高温煅烧生成水泥熟料 |
| 熟料冷却 | 高温熟料快速冷却，提高熟料强度 |
| 熟料储存 | 熟料储存均化，稳定质量 |
| 水泥粉磨 | 熟料+石膏+混合材共同粉磨为水泥成品 |
| 水泥储存 | 水泥成品储存，防止受潮 |
| 水泥包装发运 | 袋装/散装水泥装车外运 |

### 3. 原料/物料（material）
- **颜色**：紫色（#a855f7）
- **节点大小**：中等（半径 0.85）
- **实体列表**：

| 实体名称 | 描述 |
|---------|------|
| 石灰石 | 主要钙质原料，CaO来源，占原料80%以上 |
| 黏土 | 提供Al₂O₃、SiO₂，调节原料成分 |
| 石膏 | 调节水泥凝结时间，掺量3-5% |
| 混合材 | 矿渣、粉煤灰等，降低成本改善性能 |

- **颜色**：粉色（#ff6b9d）
- **节点大小**：较大（半径 1.0）
- **实体列表**：

| 实体名称 | 描述 |

| 破碎机 | 颚式、锤式、反击式破碎机，粗碎中碎细碎 |
| 立磨 | 生料/水泥粉磨核心设备，高效节能 |


## 五、工艺知识要点

### 两磨一烧
水泥生产的核心工艺可概括为"两磨一烧"：
1. **生料粉磨**（第一磨）：将原料研磨成生料粉末
2. **回转窑煅烧**：1450℃高温煅烧生成熟料
3. **水泥粉磨**（第二磨）：熟料+石膏+混合材粉磨成水泥

### 质量控制关键点
- 原料预均化：降低成分波动
- 生料均化：保证成分稳定
- 熟料冷却：快速冷却提高强度
- 熟料储存：均化稳定质量

### 主要设备配置
- 破碎系统：颚式/锤式/反击式破碎机
- 粉磨系统：立磨（生料+水泥）、球磨机（水泥终粉磨）
- 煅烧系统：回转窑+预热分解炉
- 冷却系统：第四代篦冷机
