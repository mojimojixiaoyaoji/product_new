# 水泥生产工艺知识图谱（完整版）

## 一、实体类型（Entity Types）

共 **11 大类实体**，构成工厂-产线-工序-系统-设备-测点-原料-指标-产品-质检的完整层级结构：

| 实体类型 | 代码 | 说明 | 层级 |
|---------|------|------|------|
| 工厂 | factory | 水泥生产企业的整体组织 | L1 |
| 产线 | production_line | 完整的生产线，如熟料生产线 | L2 |
| 工序 | operation | 生产过程中的主要工艺环节 | L3 |
| 系统 | system | 支撑工序运行的设备系统 | L4 |
| 设备类别 | equipment_category | 同类设备的分类 | L5 |
| 设备 | equipment | 具体的生产设备实例 | L6 |
| 测点 | point | DCS系统中的监测点位 | L7 |
| 原料 | raw_material | 生产所用的原材料 | - |
| 指标 | indicator | 工艺质量技术指标 | - |
| 产品 | product | 产出的熟料/水泥产品 | - |
| 质检 | quality_check | 质量检测记录和结果 | - |

---

## 二、关系类型（Relation Types）

共 **9 种标准关系**，描述实体间的关联：

| 关系代码 | 关系名称 | 说明 | 适用层级 |
|---------|---------|------|---------|
| include | 包含 | 上级实体包含下级实体 | 工厂→产线→工序→系统→设备类别→设备 |
| belong_to | 隶属于 | 下级实体隶属于上级实体 | 反向关系 |
| flow_to | 工艺流向 | 工序间的生产工艺流向 | 工序→工序 |
| equip_for | 设备支撑 | 设备服务于工序/系统 | 设备→工序/系统 |
| has_point | 含有测点 | 设备关联的监测点位 | 设备→测点 |
| use_material | 使用原料 | 工序/系统使用原料 | 工序→原料 |
| produce_product | 产出产品 | 工序产出产品 | 工序→产品 |
| has_indicator | 关联指标 | 工序/产品关联质量指标 | 工序/产品→指标 |
| check_quality | 质量检测 | 质检记录关联产品/原料 | 质检→产品/原料 |
