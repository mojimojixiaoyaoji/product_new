# 智能问数前端视觉和交互规范

## 1. 视觉规范

### 1.1 颜色方案
- **主色调**：
  - 主色：#D9000F（红色）- 用于标题、按钮、强调元素
  - 主色hover：#C0000D - 用于按钮悬停效果
  - 主色light：#FFF2F2 - 用于背景和高亮
- **中性色**：
  - 背景色：#F7F8FA - 页面背景
  - 卡片色：#FFFFFF - 内容卡片
  - 边框色：#E5E6EB - 分割线、边框
- **文字色**：
  - 主文字：#1D2129 - 主要标题和内容
  - 次文字：#4E5969 - 次要内容
  - 辅助文字：#86909C - 辅助信息和时间

### 1.2 布局结构
- **整体布局**：
  - 顶部导航栏：高度60px，红色背景
  - 左侧对话区：宽度420px，白色背景，带边框
  - 右侧结果区：自适应宽度，浅灰色背景
  - 中间分隔线：可调整宽度
- **左侧对话区**：
  - 历史会话栏：高度约40px
  - 历史会话列表：最大高度160px，可滚动
  - 对话标题栏：高度约40px
  - 消息区域：自适应高度，可滚动
  - 输入栏：高度约60px

### 1.3 字体和排版
- **字体**：Microsoft YaHei, sans-serif
- **字号**：
  - 标题：18px（导航栏），15px（卡片标题）
  - 正文：14px（消息内容），13px（历史会话）
  - 辅助文字：12px（时间），11px（消息时间）
- **行高**：消息内容1.5倍行高，报告内容1.8倍行高

### 1.4 组件设计
- **按钮**：
  - 主按钮：红色背景，白色文字，圆角8px
  - 次按钮：白色背景，红色边框，圆角6px
  - 图标按钮：圆形，红色背景，白色图标
- **输入框**：
  - 样式：带边框，圆角8px
  - 高度：44px
  - 内边距：12px
- **卡片**：
  - 样式：白色背景，圆角12px，带轻微阴影
  - 内边距：24px
  - 边框：左侧4px红色边框（查询卡片）
- **消息气泡**：
  - 用户消息：红色背景，白色文字，圆角16px，右下角4px
  - 机器人消息：浅灰色背景，灰色文字，圆角16px，左下角4px
- **图表**：
  - 容器：白色背景，圆角8px，带边框
  - 高度：300px
  - 内边距：16px

## 2. 交互规范

### 2.1 会话管理
- **新建会话**：点击"新对话"按钮创建新会话
- **切换会话**：点击历史会话列表中的会话项
- **删除会话**：点击会话项右侧的删除按钮，确认后删除
- **会话命名**：新会话默认名称为"新对话"，发送第一条消息后自动更新为消息内容（最多12个字符）

### 2.2 消息交互
- **发送消息**：
  - 输入框输入文本，点击"发送"按钮
  - 支持回车键发送
  - 支持语音输入（点击语音按钮）
- **消息反馈**：
  - 用户消息显示为红色气泡，右侧对齐
  - 机器人消息显示为灰色气泡，左侧对齐
  - 消息显示发送时间
- **消息选中**：
  - 点击用户消息，对应结果卡片会高亮显示
  - 点击结果卡片，对应消息会高亮显示

### 2.3 结果展示
- **查询卡片**：
  - 每条查询生成一个卡片
  - 卡片包含查询问题、时间、结果内容
  - 支持点击选中，显示高亮效果
- **内容类型**：
  - 数据表格：带表头，支持导出
  - 图表：支持线图、柱图、饼图切换
  - 分析报告：结构化文本，支持下载
- **下一步建议**：
  - 基于当前查询类型，生成相关的下一步查询建议
  - 点击建议可直接发送查询

### 2.4 图表交互
- **图表类型切换**：
  - 支持线图、柱图、饼图三种类型切换
  - 点击图表类型按钮进行切换
  - 切换时图表会重新渲染
- **图表导出**：
  - 点击"导出图表"按钮可导出图表

### 2.5 响应式设计
- **布局调整**：
  - 左侧对话区宽度可通过中间分隔线调整
  - 支持窗口大小变化时的自适应布局
- **滚动处理**：
  - 历史会话列表、消息区域、结果区域均支持滚动
  - 新消息会自动滚动到底部

### 2.6 动画效果
- **按钮悬停**：按钮背景色变化，轻微上浮效果
- **卡片选中**：卡片阴影增强，轻微上浮效果
- **分隔线**：鼠标悬停时显示红色指示线
- **消息发送**：消息气泡平滑出现
- **结果加载**：模拟加载过程，延迟显示结果

## 3. 功能规范

### 3.1 核心功能
- **智能问答**：支持自然语言查询
- **数据查询**：根据查询生成数据表格
- **图表生成**：自动生成趋势图表
- **报告生成**：基于数据生成分析报告
- **语音输入**：支持语音识别输入
- **会话记忆**：保存历史会话和查询结果

### 3.2 示例查询
- 查询最近一周的水泥强度预测情况
- 生成项目工时图表
- 撰写近一年模型更新情况数据报告

### 3.3 错误处理
- **无结果提示**：当查询无结果时，显示提示信息和建议查询
- **会话管理**：删除会话时进行确认
- **输入验证**：空输入时不发送消息

## 4. 技术实现

### 4.1 前端技术
- **基础技术**：HTML5 + CSS3 + JavaScript
- **图表库**：ECharts 5.4.3
- **存储**：localStorage（存储会话数据）

### 4.2 数据结构
- **会话数据**：
  - 会话ID、名称、时间
  - 消息列表
  - 结果卡片列表
- **查询结果**：
  - 数据表格
  - 图表配置
  - 分析报告

### 4.3 性能优化
- **图表渲染**：延迟渲染图表，避免阻塞页面
- **滚动优化**：使用requestAnimationFrame优化滚动
- **内存管理**：及时清理未使用的图表实例

## 5. 设计原则

### 5.1 用户体验
- **简洁直观**：界面简洁，操作流程清晰
- **响应迅速**：模拟实时响应，提供即时反馈
- **智能引导**：提供下一步建议，引导用户操作
- **一致性**：保持界面元素风格一致

### 5.2 视觉设计
- **层次分明**：通过颜色、阴影、边框等元素创建视觉层次
- **重点突出**：使用主色调突出重要元素
- **留白合理**：适当的留白增强可读性
- **动效自然**：动画效果流畅自然，不干扰用户操作

### 5.3 交互设计
- **操作简单**：减少操作步骤，提高效率
- **反馈及时**：操作后及时给出反馈
- **容错性**：提供错误提示和恢复机制
- **可访问性**：支持键盘操作和屏幕阅读器
